## Description
With the help of Natural Language Processing - or short NLP - we are able to evaluate an given Text. NLPs leverage machine learning and deep learning create a program that can interpret natural human speech.

In addition to this, this project displays the useage of Webpack, loaders, plugins, Service Workers and Plugins. 

## Usage
To run this code on your machine you simply need to download or clone this repository.  
 __Please be advised that this app accepts only English!!!__

## Setup 
To run this app you need to 
- Download or clone this repository
- run ```npm install```
- run ```npm run build-dev``` to create the developement build
- run ```npm run build-prod``` to create the production build

Have fun!
